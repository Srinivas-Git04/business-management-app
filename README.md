# 🚗 DSK DriveMate

A full-stack driver booking platform that connects customers with professional drivers.

## Features

- Customer Authentication
- Admin Dashboard
- Driver Management
- Driver Assignment
- Booking Management
- Customer Dashboard
- Ride Completion
- Revenue Dashboard
- Search & Filter
- Responsive Design

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL

## Installation

```bash
git clone https://github.com/yourusername/DSK-DriveMate.git

cd DSK-DriveMate

npm install

npm run dev
```

Create `.env.local`

```env
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
```
